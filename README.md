# Pool Test Advisor

A simple web app for tracking swimming pool water test results and receiving chemistry recommendations. Updated V2 deployment.

## Features

- Water test logging
- Automatic pool chemistry recommendations
- Chlorine dosage calculations
- Historical test tracking
- Delete individual history entries
- CSV export
- Progressive Web App (PWA) support
- Mobile-friendly interface

## Pool Setup

Default pool volume: 20,000 gallons

## Parameters Tracked

- Free Chlorine (FC)
- Combined Chlorine (CC)
- pH
- Total Alkalinity (TA)
- Calcium Hardness (CH)
- Cyanuric Acid (CYA)

## How to Use

1. Enter current test results.
2. Click **Analyze & Save**.
3. Review recommendations and dosage guidance.
4. Track trends in the History section.
5. Export data anytime as CSV.

## Disclaimer

Recommendations are estimates only. Always verify chemical additions before treating pool water.
